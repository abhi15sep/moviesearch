# Movie Search

To fast install dependences, I recommend to use [yarn](https://yarnpkg.com/lang/en/). Otherwise, you can use `npm run` instead `yarn`.

For example:
    `npm i && npm run dev`
    `npm run test`
    `npm run build`

## Install 

`yarn` 

## Start

`yarn dev`

## Build
Your build version will be under `dist` folder.

`yarn build`

## Test

`yarn test`
